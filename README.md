# FoodGo - Food Delivery Website

FoodGo is a responsive food delivery website inspired by modern online food ordering platforms.

## Features

- Responsive design
- Restaurant cards
- Search functionality
- Category filter
- Add to cart
- Remove from cart
- Checkout alert
- Mobile menu
- Special offers section

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
